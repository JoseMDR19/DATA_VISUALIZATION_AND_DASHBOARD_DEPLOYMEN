## Objetivo
Diseñar y desplegar un **dashboard analítico interactivo** que muestre información sobre:
- Tasas de retención de estudiantes
- Niveles de satisfacción
- Comparación entre periodos académicos (Spring/Fall)

El propósito es ayudar a la universidad a analizar y mejorar sus procesos de **admisión, retención y satisfacción estudiantil**.
